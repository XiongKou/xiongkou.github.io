# vue基本操作

```js
npm uninstall后面接要 删除的内容

npm i 每次修改了pack.age.json里面的依赖的内容。都要更新下

npm run serve/dev 开发本地环境

ctrl+c停止脚手架服务

脚手架运行起来后，本机地址要开通端口号
控制面板>系统和安全>windows defender 防火墙>高级设置>入站规则>新建规则>端口>特定本地端口http://192.168.0.100:8080/其中的端口号只是8080

vue的结构，属性定义在data里面，methods里面定义的都是方法，computed里面定义的方法计算属性，调用的方法和属性一样


```
